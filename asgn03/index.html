Assignment 3

For this assignment I decided to build off of my previous game. (jump to game)

I retained all the features of the previous game and added some new ones.

Here is a list of new features added:

  1. Pause Menu which shows directions for playing the game and allows you to select a difficulty level (with its own background)
  2. An Easy and a Hard level. The level changes the types of barriers and the speed of the player
  3. Background music which loops in the background
  4. Barriers in the game. (The easy level barriers are a little easier to avoid than the hard level barriers)
  5. Hitting a barrier triggers a "boing" noise
  6. Made use of methods to keep code more organized. The draw function just has a few simple method calls and all the logic is contained in those methods.
