Final Project Update 4

We have made good progress on the project and are getting close to being finished. Since the last update we have made the tile selection menu work so that the tiles can actually be placed into the level creator. A save system has bee implemented and on game start the user is able to select which level they would like to play. This is accomplished using file IO. Additionally the game has been made more generic to support more features. The "?" black has been reworked to be more dynamic and to make the coin and mushroom easy to distinguish. There is also a castle at the end of created levels and when the game is actually played it is able to run the game end sequence making the custom levels "win-able". There has also been another breakable block added.

Mostly all that remains is ironing out some bugs and creating the physical controller. This will require remapping some of the keys we are using in the level creator but should not be too hard overall.
