Final Project Update 3

We have made good progress on the project. There is a menu of blocks which can be selected and placed into the level. The work in progress is included below.

  • From the main menu press c to access the level creator.
  • Then you can use the arrow keys to navigate tiles.
  • Use the space bar to bring up the menu and the arrow keys to navigate the menu.
  • Use the enter key to select a tile and the backspace key to go back.
  • Once you selected a tile use enter to place it at the cursor.
